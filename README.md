# 十五音輸入法

本十五音輸入法，所採用之聲母、韻母，以《彙集雅俗通十五音》為
標準。而聲母與韻母所採用之編碼，同台羅拼音。

## 輸入方案特性簡介

本專案發展之輸入法，及其功能特性說明如下。

- **ngsth_bun / 雅俗通·文** - 以台羅拼音輸入漢字。輸入過程中
  ，選字視窗會顯示十五音呼切標音。輸入法僅提供雅俗通之文讀音
  漢字。
- **peh_ue_im / 台灣．白話音** - 以台羅拼音輸入漢字。輸入過
  程中，選字視窗會顯示十五音呼切標音。輸入法之漢字不僅有雅俗
  通 之文讀音；亦有台灣使用之白話音。
- **tsap_goo_im / 十五音** - 以十五音呼切法輸入漢字。輸入過
  程中，需依（韻母）、（調號）及（聲母）之順序輸入拼音碼。輸
  入法所提供之漢字，其拼音含：文讀音與白話音。

## 單字／詞彙／俗諺漢字庫簡介

- **ngsth_bun.dict.yaml** - 《雅俗通．文》輸入方案使用之單字
  庫
- **peh_ue_im.dict.yaml** - 《台灣．白話音》輸入方案使用之單
  字庫
- **peh_ue_im_su_khoo.dict.yaml** - 《台灣．白話音》輸入方案
  使用之詞彙庫
- **peh_ue_im_siok_gi.dict.yaml** - 《台灣．白話音》輸入方案
  使用之俗諺庫
- **peh_ue_im_ka_ki.dict.yaml** - 《台灣．白話音》輸入方案使
  用之個人專用單字／詞彙庫
- **tsap_goo_im.dict.yaml** - 《十五音》輸入方案使用之單字庫
  （尚在開發中）
